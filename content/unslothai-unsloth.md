# unslothai/unsloth

2x faster fine-tuning, 60% less VRAM usage. Web UI for training (Unsloth Studio). Fine-tune open-source models on custom datasets without a cluster.





## What It Does
Unsloth Studio — a web UI for training and running open-source models like Gemma 4, Qwen 3, Llama 4, Phi 4, Mistral, and DeepSeek. Makes fine-tuning accessible with a visual interface. Also provides the core Unsloth library for 2x faster training with 60% less memory.

## Key Features
- 2x faster fine-tuning, 60% less VRAM usage
- Web UI for training (Unsloth Studio)
- Supports: Gemma 4, Qwen 3, Llama 4, Phi 4, Mistral, DeepSeek
- LoRA, QLoRA, and full fine-tuning
- One-click training pipeline
- Model evaluation and comparison tools
- GGUF export for local inference

## Tech Stack
- Language: Python
- License: Apache-2.0

## Why I Starred It
Fine-tuning used to require deep ML knowledge and expensive compute. Unsloth makes it accessible. The 2x speed + 60% memory reduction is the real differentiator.
