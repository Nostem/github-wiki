## What It Does
Community platform for sharing, discovering, and collecting prompts for ChatGPT, Claude, Gemini, and other LLMs. Previously known as Awesome ChatGPT Prompts. Now a full web app with prompt collections, voting, and categorization.

## Key Features
- Thousands of curated prompts across categories (writing, coding, business, creative)
- Multi-model support: ChatGPT, Claude, Gemini, Llama
- Community voting and prompt quality ranking
- Prompt collections and bookmarking
- Copy-to-clipboard one-click usage
- Category browsing and search

## Tech Stack
- Language: HTML
- License: MIT

## Why I Starred It
The largest prompt community. When I need a prompt for a specific task, this is better than guessing — someone has probably already optimized it.
