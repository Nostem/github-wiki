# awesome-claude-code-subagents

A collection of 100+ specialized Claude Code subagents covering a wide range of development use cases.

## What It Does
Pre-built subagents for Claude Code — each subagent is a focused agent configured for a specific task domain (e.g., frontend developer, DevOps engineer, security auditor). Rather than writing a generic agent prompt, you pick a subagent that knows its domain.

## Key Features
- 100+ specialized subagents
- Domain-specific configurations
- Ready to use with Claude Code
- Covers development, DevOps, security, and more

## Tech Stack
- Language: Shell (subagent definitions)
- Platform: Claude Code

## Why I Starred It
Subagents solve the "jack of all trades, master of none" problem. When we need a specific capability, a specialized agent configured for that domain performs better than a generalist. This library shows how to structure that specialization.
