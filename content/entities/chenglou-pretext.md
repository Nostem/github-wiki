# chenglou/pretext

## What It Does
Fast, accurate, and comprehensive text measurement and layout engine. Precisely measures text dimensions, line breaks, and layout constraints for any font and size. Used for building text editors, IDEs, and any UI that needs pixel-perfect text rendering.

## Key Features
- Pixel-accurate text measurement
- Line break and word wrap calculation
- Font metrics extraction
- Supports any font and size
- High performance — handles large documents
- TypeScript API

## Tech Stack
- Language: TypeScript
- License: MIT

## Why I Starred It
Text measurement is harder than it looks. If you need pixel-perfect text layout, this handles the edge cases that naive approaches miss.
