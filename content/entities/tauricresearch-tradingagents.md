## What It Does
Multi-Agent LLM Financial Trading Framework. Uses specialized agents (analyst, trader, risk manager, portfolio manager) that debate and collaborate to make trading decisions. Each agent has a distinct role and perspective, producing more robust decisions than single-agent approaches.

## Key Features
- Multi-agent debate: analyst, trader, risk manager, portfolio manager
- Role-based agent specialization for financial decisions
- Agent debate and consensus mechanisms
- Multi-source data integration (news, price, fundamentals)
- Backtesting framework
- Risk management agent enforces position limits

## Tech Stack
- Language: Python
- License: MIT

## Why I Starred It
Multi-agent debate for trading decisions is a better architecture than single-agent. The role separation (analyst vs risk manager) prevents confirmation bias.
