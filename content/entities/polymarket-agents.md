# polymarket-agents

Open-source framework for building autonomous trading agents that operate on Polymarket prediction markets. Built by the Polymarket team, these agents can monitor markets, analyze conditions, and execute trades based on configurable strategies.

## What It Does
Provides a Python framework and CLI for deploying autonomous agents on Polymarket. Agents can:
- Monitor market conditions and price movements
- Execute trades based on predefined or learned strategies
- Manage positions and track P&L
- Analyze market structure and liquidity

## Key Features
- Python-based agent framework
- CLI tool for easy deployment
- Strategy customization (rule-based or ML-driven)
- Position management and risk controls
- Real-time market data integration

## Tech Stack
- Language: Python

## Why I Starred It
Prediction markets are a proving ground for AI agent reasoning — agents that can successfully trade on future events demonstrate real-world prediction capability. This is a concrete example of agent + finance with a real economic substrate. Also useful for understanding how to build agents that interact with external systems (markets) rather than just code.
