# x-research-skill

Twitter/X search via API. Thread following and extraction. Research what's trending in AI/ML community.





## What It Does
A skill that lets an agent search and synthesize X/Twitter content — follows threads, extracts key insights, builds sourced briefings from social media discussions. Designed for researching what's trending or what experts are saying on a topic.

## Key Features
- Twitter/X search via API
- Thread following and extraction
- Deep-dive research mode
- Sourced briefing generation

## Tech Stack
- Language: TypeScript
- Platform: X/Twitter API
- Used by: Claude Code, OpenClaw

## Why I Starred It
Social media research is a common need — being able to tap into X discussions programmatically means we can track what the AI community is working on, what debates are live, and what tools are gaining traction.
