# tw93/Pake

Turn any webpage into a lightweight desktop app with a single command. Uses Rust/Tauri under the hoo

## What It Does
Turn any webpage into a lightweight desktop app with a single command. Uses Rust/Tauri under the hood for tiny bundle sizes (~5MB vs Electron's ~100MB). Supports custom window sizing, shortcuts, and always-on-top. Much faster and lighter than Electron alternatives.

## Key Features
- One command to convert any URL to desktop app
- ~5MB bundle size (vs ~100MB Electron)
- Rust/Tauri backend — fast and memory-efficient
- Custom window size, title bar, shortcuts
- Always-on-top mode
- Multi-platform: macOS, Windows, Linux
- CLI and GitHub Action for automated builds

## Tech Stack
- Language: Rust
- License: MIT

## Why I Starred It
Electron is bloated. Pake gives you native desktop apps from web pages at 1/20th the size. Perfect for wrapping our web dashboards.
