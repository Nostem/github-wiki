## What It Does
Open-source orchestration platform for 'zero-human companies' — autonomous businesses run entirely by AI agents. Handles task routing, agent coordination, quality control, and human-in-the-loop escalation. Designed to run companies with minimal human oversight.

## Key Features
- Agent orchestration for autonomous business operations
- Task routing and assignment to specialized agents
- Quality control and output validation
- Human-in-the-loop escalation for critical decisions
- Multi-agent coordination framework
- TypeScript-based, containerized deployment

## Tech Stack
- Language: TypeScript
- License: MIT

## Why I Starred It
The 'zero-human company' concept is extreme but the orchestration patterns are useful for any multi-agent system. Good reference for agent coordination.
