# jaywcjlove-awesome-mac

500+ quality-filtered macOS apps across 20+ categories. Emphasis on open-source and free software. Find native macOS alternatives to existing tools when switching platforms.




## What It Does
The largest curated directory of high-quality macOS software, organized into categories: Reading/Writing Tools (text editors, markdown, ebooks, RSS), Developer Tools (IDEs, version control, virtualization, databases, hybrid frameworks), Terminal/CLI apps, Design/Prototyping, Communication/Team Tools, System Utilities (clipboard management, window tiling, security, Finder enhancements), and a growing AI Tools section. Uses a legend system: OSS (open source), Free, App Store, Native, and Awesome (links to sub-lists). Available in English, Chinese, Korean, and Japanese. Strictly anti-piracy.

## Key Features
- 500+ quality-filtered macOS apps across 20+ categories
- Emphasis on open-source and free software
- Icon legend: OSS, Free, App Store, Native, Awesome
- Developer Tools section: IDEs, API tools, network analyzers, regex editors, Git clients, Docker/VM clients, database GUIs
- System Utilities: clipboard managers, window tiling, cleanup tools, VPN/password managers, Finder enhancements
- AI Tools section (growing)
- Multilingual: English, Chinese, Korean, Japanese
- Anti-piracy stance with blocked site list
- Regularly updated with community contributions

## Tech Stack
- Language: Swift (for associated tooling)

## Why I Starred It
When setting up a new Mac or looking for a native alternative to a cross-platform tool, this is the best starting point. The quality filtering matters — it's not just a dump of every app, only high-quality ones make the cut. Useful for finding macOS-native dev tools that feel right on the platform.
