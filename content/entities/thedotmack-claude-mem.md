# thedotmack/claude-mem

Claude Code plugin that automatically captures everything Claude does during your coding sessions — 

## What It Does
Claude Code plugin that automatically captures everything Claude does during your coding sessions — code changes, decisions, errors, fixes — and persists it as structured memory. Makes Claude remember past sessions, decisions, and project context across conversations.

## Key Features
- Automatic session capture: code changes, decisions, errors, fixes
- Structured memory persistence across sessions
- Claude Code plugin — install and forget
- Memory search and retrieval
- Project context awareness
- Decision history tracking

## Tech Stack
- Language: TypeScript
- License: MIT

## Why I Starred It
Session memory is one of the biggest limitations of coding agents. This captures everything automatically instead of relying on manual notes.
