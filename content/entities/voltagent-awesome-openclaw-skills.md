# awesome-openclaw-skills

The awesome collection of OpenClaw skills — 5,400+ skills filtered and categorized from the official OpenClaw Skills Registry.

## What It Does
A curated, categorized index of OpenClaw skills. If you want to know what capabilities exist for OpenClaw agents, this is where to look. Skills cover everything from browser automation to code execution to data processing.

## Key Features
- 5,400+ skills catalogued
- Categorized by domain and capability
- Links to official skill registry
- Filterable by use case

## Tech Stack
- Language: TypeScript (skill definitions)
- Platform: OpenClaw

## Why I Starred It
Massive reference for what's possible with agentic systems. When we need a specific capability — say, PDF processing or code execution — this tells us if a skill already exists and what it's called.
