# claude-code-extensions

Skills, best practices, and extensions for Claude Code — the ecosystem of tools, prompts, and frameworks that extend Anthropic's official CLI coding agent into a more powerful, specialized, or workflow-integrated system.

## Definition
Claude Code is the official Anthropic CLI for agentic coding. The extension ecosystem around it includes skill collections (curated tool-use instructions), best practice guides, harness optimization systems, and specialized sub-agents. This ecosystem is one of the fastest-growing in the AI agent space.

## Key Extensions
- **Skill collections**: Focused tool-use instructions for specific domains
- **Harnesses**: Optimize Claude's behavior through system prompts
- **Instincts**: Low-level behavioral patterns baked into the agent
- **Memory**: Persistent context across sessions
- **Spec-driven**: Context engineering through spec documents
