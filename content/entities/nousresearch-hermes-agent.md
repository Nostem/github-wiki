# nousresearch-hermes-agent

"The agent that grows with you" — a self-improving agent framework from Nous Research. Hermes is designed to learn from its interactions, improve its strategies over time, and adapt to the user's workflow rather than requiring the user to adapt to it.

## What It Does
An agent framework built on OpenClaw that emphasizes self-improvement and adaptability. Key difference from static agents: Hermes observes its own performance, identifies failure modes, and adjusts its approach. Built for developers who want an agent that gets better the more they use it.

## Key Features
- Self-improving: learns from past mistakes
- OpenClaw foundation (interoperable with other OpenClaw tools)
- Strategy adaptation to user preferences
- Tool-use and skill composition
- CLI-first design

## Tech Stack
- Language: Python
- Framework: OpenClaw

## Why I Starred It
The self-improvement angle is different from most agents that just execute and report. If Hermes actually learns from interactions (not just context), it's a step toward agents that require less supervision over time. Nous Research has a reputation for strong fine-tuning work — this extends that into agentic territory.
